QUESTIONÁRIO
Questão 1: Explique qual é a função de um servidor web em um ambiente de hospedagem e cite pelo menos um exemplo.

Resposta: Um servidor web lida com requisições HTTP/HTTPS e serve conteúdo estático (HTML, CSS, JS, imagens). Sua função é entregar arquivos prontos sem processamento adicional. Exemplo: Apache HTTP Server (código aberto e amplamente utilizado).
Questão 2: Qual a principal característica do NGINX e por que ele é muito utilizado em projetos de alto desempenho?

Resposta: Característica: Leveza e eficiência. É otimizado para alto desempenho, atua como proxy reverso e consome menos recursos, ideal para tráfego intenso.
Questão 3: Diferencie os protocolos HTTP e HTTPS. Por que o HTTPS é recomendado para aplicações modernas?

Resposta: HTTP: Protocolo sem criptografia (dados trafegam em texto claro, como foram escritos).
HTTPS: Versão segura com criptografia SSL/TLS (protege dados contra interceptação). Importância do HTTPS: Previne roubo de dados (como senhas), garante autenticidade.
Questão 4: Qual é a função das portas em uma rede de computadores e qual a importância da porta 443?

Resposta: Função:Canais de comunicação que direcionam tráfego a serviços específicos. Porta 443: Padrão para HTTPS. Importância: Permite conexões seguras e criptografadas em sites e aplicações.
Questão 5: Defina o conceito de endereço IP e explique sua importância em redes de computadores.

Resposta: Definição: Número único que identifica dispositivos em uma rede (ex.: 192.168.1.1).
Importância: Permite a comunicação entre dispositivos e o roteamento correto de dados na internet.
Questão 6: Explique o que é hospedagem VPS e em quais situações ela é mais indicada.

Resposta: O que é: Servidor Virtual Privado (parte de um servidor físico isolada para um usuário).
Indicações: Projetos com tráfego médio, necessidade de personalização (ex.: instalar softwares específicos) ou quando a hospedagem compartilhada é insuficiente.
Questão 7: Descreva as vantagens do servidor web Caddy em relação a outros servidores tradicionais.

Resposta: Vantagens: Configuração simplificada (ex.: HTTPS automático com certificados gratuitos). Moderno e adequado para implantação rápida, sem complexidade de configuração manual.
Questão 8: Qual servidor de aplicação é mais indicado para aplicações desenvolvidas na plataforma .NET (C#)? Explique o motivo.

Resposta: ASP.NET Core (plataforma da Microsoft).
Motivo: Projetado para aplicações web e APIs em C#, com integração nativa ao ecossistema .NET.
Questão 9: Descreva o funcionamento do modelo Cliente-Servidor no contexto de sistemas web.

Resposta: Cliente (navegador) envia requisições (ex.: carregar uma página). O servidor processa a solicitação (entrega conteúdo estático ou dinâmico) e retorna uma resposta.
Questão 10: Explique o que é o Apache Tomcat e para qual tipo de aplicação ele é mais indicado.

Resposta: O que é: Servidor de aplicação que executa servlets e JSP.
Indicação: Aplicações Java EE (ex.: sistemas corporativos com lógica dinâmica).
Questão 11: Diferencie, com suas palavras, um servidor web de um servidor de aplicação, destacando suas funções principais.

Resposta: Servidor Web é um servidor que entrega arquivos estáticos.
Servidor de aplicação: é o que está por trás do servidor web, executa a lógica da aplicação e entrega respostas dinâmicas.
Questão 12: Defina o conceito de domínio na internet e sua relação com os endereços IP.

Resposta: Definição: Nome legível que representa um endereço IP (ex.: www.google.com).
Relação com IP: Traduz nomes amigáveis em números IP para acesso a servidores.
